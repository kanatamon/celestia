How to install

Celestia is pending Chrome Web Store review. In the meantime, install it manually in under a minute.

  1. Click Download Extension above and unzip the file.
  2. Open Chrome and go to chrome://extensions.
  3. Enable Developer mode using the toggle in the top-right corner.
  4. Click Load unpacked and select the unzipped dist folder.
  5. Celestia appears in your toolbar. Open a TikTok Live tab and click it.
